3. Bot Message Logging

HumanGPT may store only the messages generated by the bot itself in JSON files for moderation, security, and safety review purposes. This is done to allow administrators to review suspicious messages. Key points:

  • User messages are never stored, logged, or used for AI training.
  • Stored bot messages are accessible only to authorized administrators.
  • Data is stored securely in JSON files, not in executable files like .js.
  • Messages are retained only as long as necessary and may be periodically deleted.
  • No messages are sold, shared, or used for advertising purposes.
